# askai.kr

ASKAI 아스카이는 스크린샷을 붙여넣고 펜, 형광펜, 도형, 이름표로 빠르게 표시한 뒤 다시 복사하는 개인용 비주얼 클립보드입니다.

## Browser Extension

Chrome, Edge, Firefox 확장프로그램 소스는 `extension/src`에 있습니다. 우클릭 메뉴에서 보이는 화면, 영역 선택, 전체 페이지 캡처를 실행할 수 있습니다.

```bash
npm run extension:build
```

- Chrome / Edge 설치 폴더: `extension/dist/chrome`
- Firefox 설치 폴더: `extension/dist/firefox`
